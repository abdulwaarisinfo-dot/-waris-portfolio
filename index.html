System Design Depth
8-Layer
Architecture

The AI E-Commerce Assistant isn't a wrapped ChatGPT call — it's a purpose-built retrieval intelligence system. Eight discrete layers handle everything from WebSocket transport to behavioral analytics, working in concert to deliver sub-second, contextually aware responses across Urdu and English.

01 — Transport

WebSocket Layer

Persistent bidirectional connections. Zero re-handshake overhead per message. Real-time streaming throughout sessions.

02 — Runtime

FastAPI Async Layer

ASGI non-blocking I/O. Concurrent session management without threading overhead.

03 — Language

NLP Query Layer

Urdu + English detection. Category, color, material extraction. Intent classification via pattern matching.

04 — Memory

Session Memory

Per-session state with multi-turn context. Preference history enables contextual follow-up generation.

05 — Retrieval

Product Retrieval

NLP signal → MongoDB filter translation. Querying against category, color, material, and price constraints.

06 — Scoring

Ranking Engine

6-dimension composite score: keyword relevance, category, color, material, rating, and trending weight.

07 — Telemetry

Analytics Pipeline

Behavioral event stream: searches, clicks, views, price signals, language patterns. Live feedback loop for calibration.

08 — UX

Dynamic Suggestions

Contextual follow-ups based on session state. 2–4 relevant refinements per turn guiding discovery flow.
